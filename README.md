# Rocket Trajectory Prediction Software

## Project Purpose
This software simulates the trajectory of a multistage rocket during its ascent through the atmosphere, taking into account factors such as drag and thrust modeling. It is designed to help engineers and enthusiasts understand rocket dynamics and optimize rocket designs.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/NihalZevy/Rocket-Trajectory-Prediction-Software.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Rocket-Trajectory-Prediction-Software
   ```
3. Install required dependencies. Make sure you have Python, MATLAB, and C++ compilers installed.

## Basic Usage
- To run the simulation, use the following command in your terminal:
   ```bash
   python main.py
   ```
- Adjust the parameters in the configuration file as needed to test different scenarios.
